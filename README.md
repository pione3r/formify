## 개요
- 다양한 질문 유형의 설문조사를 만들 수 있는 웹서비스

## 서비스 사용 방법
- 아래 URL의 웹페이지 접속 후, 로그인
- 만들고 싶은 질문폼을 생성한 후, 응답자들에게 링크 공유
- 응답자들은 해당 링크에 접속하여 질문 응답
- 응답받은 결과 조회

## URL
- 웹서비스 URL : https://formify-pione3r.vercel.app/

- Next.js
